# SME Ops 系統教學手冊

## Part 1：員工端（LINE LIFF）

---

### 1-1 打卡上下班

**操作步驟：**
1. 打開 LINE → 點選 SME Ops 官方帳號
2. 點選 Rich Menu 的「打卡」
3. 系統自動偵測 GPS 位置
4. 確認位置正確 → 點「打卡」按鈕
5. 顯示打卡成功（時間、地點、IP）

**注意事項：**
- 需在門市 150 公尺範圍內
- 支援 GPS + WiFi 雙重驗證
- 忘記打下班卡 → 系統凌晨 6:00 自動推播提醒

> 📸 截圖：LIFF 打卡畫面（打卡前 + 打卡成功）

---

### 1-2 查看我的班表

**操作步驟：**
1. LINE → SME Ops → 點「我的班表」
2. 顯示本週/本月排班
3. 可看到班別時間、休假日

> 📸 截圖：LIFF 我的班表畫面

---

### 1-3 請假申請

**操作步驟：**
1. LINE → SME Ops → 點「請假」
2. 選擇假別（特休/病假/事假...）
3. 選擇日期範圍
4. 填寫請假原因
5. 點「提交」
6. 系統自動通知主管審核

**假別說明：**
| 假別 | 說明 | 薪資 |
|------|------|------|
| 特休 | 依年資計算天數 | 全薪 |
| 病假 | 需附證明 | 半薪 |
| 事假 | 個人事務 | 無薪 |
| 婚假 | 8天 | 全薪 |
| 喪假 | 依親等3-8天 | 全薪 |

> 📸 截圖：LIFF 請假申請畫面 + 請假列表

---

### 1-4 加班申請

**操作步驟：**
1. LINE → SME Ops → 點「加班申請」
2. 選擇日期、填寫加班時數
3. 填寫加班原因
4. 提交 → 等待主管審核

> 📸 截圖：LIFF 加班申請畫面

---

### 1-5 補打卡申請

**適用情況：** 忘記打卡、系統異常

**操作步驟：**
1. LINE → SME Ops → 點「補打卡」
2. 選擇日期
3. 選擇類型（上班/下班）
4. 填寫正確時間
5. 填寫原因
6. 提交 → 等待主管審核

**自動提醒機制：**
- 每天凌晨 6:00 系統自動掃描
- 有上班卡但沒下班卡 → LINE 推播提醒

> 📸 截圖：LIFF 補打卡畫面 + LINE 未打卡提醒通知

---

### 1-6 查薪水

**操作步驟：**
1. LINE → SME Ops → 點「查薪水」
2. 選擇月份
3. 顯示薪資明細（底薪、津貼、加班費、扣款、實發）

> 📸 截圖：LIFF 薪資明細畫面

---

### 1-7 費用申請

**流程：先申請 → 核准後購買 → 買完核銷**

**申請步驟：**
1. LINE → SME Ops → 點「費用申請」
2. 選擇「費用」或「非費用」
3. 選擇會計科目
4. 填寫品項明細（品名、數量、單價）
5. 可上傳訂購單/報價單（拍照）
6. 系統自動計算合計金額
7. 提交 → 依金額走對應簽核鏈

**核銷步驟：**
1. 核准後 → 進行採購
2. 買完後回到費用申請 → 點「核銷」
3. 填寫實際金額
4. 上傳收據/發票
5. 提交 → 財務確認

**金額分級簽核：**
| 金額 | 簽核流程 |
|------|---------|
| ≤ $3,000 | 主管審核 |
| $3,001 ~ $10,000 | 主管 → 部門主管 |
| > $10,000 | 主管 → 部門主管 → 財務 |

> 📸 截圖：LIFF 費用申請（品項明細 + 上傳附件） + 科目分類選單

---

### 1-8 報帳

**適用情況：** 已經花費的費用報帳（出差交通費、餐費等）

**操作步驟：**
1. LINE → SME Ops → 點「報帳」
2. 選擇分類（交通/住宿/餐飲/設備/其他）
3. 填寫金額、日期、說明
4. 可上傳收據
5. 提交 → 主管審核 → 財務核銷

> 📸 截圖：LIFF 報帳畫面

## Part 2：主管端（後台系統）

---

### 2-1 登入系統

**登入方式：**
- Google 帳號登入（推薦）
- Facebook 帳號登入
- Email + 密碼登入

**登入後自動綁定：**
系統會依據登入 email 自動對應員工資料，無需額外設定。

**網址：** https://sme-ops-system.vercel.app

> 📸 截圖：登入頁面（Google/FB/Email 三個按鈕）

---

### 2-2 排班管理

**功能：**
- 月曆式排班表
- 拖拉排班
- AI 自動排班建議
- 發布班表 → LINE 通知員工

**操作步驟：**
1. 人員組織 → 出勤管理 → 排班
2. 選擇門市、月份
3. 點選日期格子 → 選擇班別
4. 排完後點「發布」→ 自動推送 LINE 通知

> 📸 截圖：排班管理頁面（月曆 + 班別選單）

---

### 2-3 審核管理

**審核項目：** 請假、加班、補打卡、費用申請

**操作步驟：**
1. 收到 LINE 通知「有新的簽核請求」
2. 進入系統 → 流程管理 → 簽核設定
3. 查看待審核項目
4. 點「核准」或「退回」（退回需填原因）

> 📸 截圖：簽核系統頁面（待簽核列表 + 核准/退回按鈕）

---

### 2-4 出勤報表

**功能：**
- 每日出勤紀錄
- 遲到/早退統計
- 班表 vs 實際出勤比對

**操作步驟：**
1. 人員組織 → 出勤管理 → 打卡追蹤
2. 選擇日期
3. 查看所有員工打卡狀態

> 📸 截圖：打卡追蹤頁面

## Part 3：管理端

---

### 3-1 組織架構

**功能：**
- 部門管理（新增/編輯/指定主管）
- 門市管理（新增門市、設定打卡範圍）
- 員工管理（新增/編輯/設定權限）
- 組織圖（自動生成層級圖）

> 📸 截圖：組織架構圖 + 員工列表 + 部門管理

---

### 3-2 專案管理

**三層架構：** Project → Workflow → Task

**操作步驟：**
1. 流程管理 → 專案
2. 可從「專案模板」一鍵部署
3. 或手動新增專案
4. 專案底下建立多個流程（平行執行）
5. 每個流程底下有多個任務
6. 進度自動計算

**專案模板：**
- 門市裝潢翻新（3 流程 / 10 任務）
- 新人到職 SOP（2 流程 / 8 任務）
- 月底門市盤點（1 流程 / 5 任務）

> 📸 截圖：專案列表（含進度圓餅圖） + 專案詳情 + 模板部署

---

### 3-3 流程管理

**功能：**
- 建立流程範本
- 部署流程到門市
- 追蹤每個步驟進度
- AI 助手建議流程

> 📸 截圖：流程管理頁面（進行中流程列表）

---

### 3-4 簽核設定

**兩種簽核：**
1. **一般簽核** — 手動指定簽核人和步驟
2. **費用簽核** — 依金額自動分級

**費用簽核設定：**
1. 流程管理 → 費用簽核設定
2. 設定金額範圍 + 簽核步驟
3. 員工提交費用申請時自動匹配

> 📸 截圖：簽核鏈設定頁面 + 費用簽核設定（金額卡片）

### 3-5 會計科目

## 常見問題 FAQ

**Q: 打卡失敗怎麼辦？**
A: 確認 GPS 已開啟，在門市 150m 範圍內。如仍失敗，提交補打卡申請。

**Q: 忘記打下班卡？**
A: 系統會在凌晨 6:00 自動推 LINE 提醒，請至補打卡申請提交。

**Q: 費用申請和報帳有什麼不同？**
A: 費用申請是「先申請後購買」，報帳是「已經花費的費用報銷」。

**Q: 如何變更員工權限？**
A: 人員組織 → 員工 → 編輯 → 系統權限下拉選單。

**Q: 專案和流程的關係？**
A: 專案包含多個流程，流程包含多個任務。流程可以平行執行。
