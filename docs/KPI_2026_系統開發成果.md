## 二、核心系統成果（依模組）

### 1. 簽核 / 流程引擎（最大宗）
建立可設定、可稽核、跨表單通用的企業級簽核鏈。
- **動態簽核鏈**：新增申請人主管三級鏈（L1/L2/L3）、找不到簽核人自動跳過（skip_if_no_approver）、HR 表單三路分流（門市/行政/部門）
- **簽核代理（代簽）**：後端比對 + 管理頁 + 全 8 種表單類型 LINE 代簽卡與進關通知
- **加簽**：申請段/驗收段可各自綁進任務步驟
- **叫貨申請單**：獨立簽核鏈（doc_type 子型別，共用費用引擎，A/B/C 三階段）
- **費用/核銷全面升級**：核銷（驗收）單位、附件上限 3→20、複製重送、批次補結歷史單（137 筆）、卡死偵測 RPC
- **簽核中心**：Web 左右主從版面、費用鏈聚合 RPC 一次回完整步驟、改 inline 逐張核准/退回＋勾選批次通過（對齊 LIFF）
- **招募錄取動態簽核鏈**：每張錄取自己挑多位簽核人依序簽核、逐關推進、簽核中心可直接簽、進度時間軸；並堵住費用頁繞過簽核鏈的後門

### 2. 任務系統
把「表單填寫」深度整合進任務流程。
- **綁定表單自己填 / 他人填**：他人填→指派 + LINE 通知；自己填→任務內原生彈窗/iframe 當場填（含費用/調撥/稽核重型表單）
- **核銷段挑單**：未綁單時可挑「待驗收」費用單（跨步驟/多組）
- **@mention 標記同事 + LINE 通知**、任務建立者顯示、變更日誌＋活動時間軸整合
- **跨部門工單（cross-dept work order）**：營運部開單請他部門辦、狀態機＋期望/排定雙日期、LIFF＋LINE 通知；與專案/流程/任務雙向整合（受理後轉、任務指派他部門＝開工單、綁定完成自動結案）
- 綁定填寫改 inline 原生彈窗（不開新分頁）、流程/專案全任務完成自動已完成（DB 觸發器）
- 班表總覽複製/貼上（Ctrl+C/V、平鋪填滿）

### 3. HR / 薪資 / 出勤
- **薪資引擎收斂**：試算＝入帳（generate 共用 _compute）、二代健保補充保費、離職特休結清、加班費休息日 deem＋國定固定 8h、行政遲到早退寬限接門市設定
- **固定投保級距制**：勞健保/勞退改吃固定投保級距（匯入 79 人）、雇主勞退 6% 可逐人開關、勞退自提 6%、健保眷屬保費接眷屬名單
- **勞健退投保管理**：投保異動時間軸（加保/退保/眷屬加保歷史）、薪資明細存/顯示分項（勞保/健保/勞退自提/所得稅/特休折現）＋加班逐筆明細
- **薪資「發布給員工」閘門**：獨立 published_at 不刪草稿，員工不再提前看到未定案薪資
- **計薪規則精修**：本薪工時封頂在班表（超時走加班申請）、離職特休折現改年資現算＋平均工資日薪、兼職特休依 6 個月實際排班比例、天災（paid）照給薪補班表時數
- **假別**：依年資/法定動態計算餘額（對齊 LIFF/104）、批次調整、產檢假 5→7 天、每月加班上限 46h 硬擋
- **出勤**：打卡管理 admin inline 改時間（原因必填＋歷史）、換天時間可後台設定＋補打卡早上 6 點才換天、行政午休固定扣 60 分、打卡核對報表
- **門市業績獎金**：自訂欄位、紀律/獎勵分級（大過/小功/大功）、兩條領取資格原則
- **證明文件**：在職/離職證明一鍵開立下載（含版面重製：logo 浮水印/A4 尺寸/公司資訊）
- **職位管理**：職稱清單改資料表可後台自編＋拖曳排序
- **代發薪**：Excel(.xlsx) 四欄匯出、薪資明細展開完整計算過程

### 4. 門市稽核
- LIFF 獨立權限 `liff.store_audit`（預設 admin+，可逐人開）
- 第三態「不扣分」標記、扣分/得分雙欄、照片附件、備註欄、稽核室全覽
- **稽核表 v2 大改**：新版 6 類 142 項、群組層級配分、評分制（每項填扣分）、群組說明常駐（對齊 Excel 合併格）、整張單共用照片（20 張）、總平均計分；Web＋LIFF 同步

### 5. POS / 餐飲 / 訂位（新模組）
- **POS 完整化**：廚房出單、桌卡列印、結帳收據、發票載具、X 報表、作廢/折扣/退款/內外帶、門市月業績報表
- **服務員點餐模式**、顧客點餐頁重設計、菜單多語言、全門市菜單匯入（7 分類 49 品項）
- **訂位模組**：4 張表 + 平面圖拖曳佈局

### 6. 組織 / 權限 / 資安
- **RBAC 細化**：10+ HR 路徑 sidebar/page gate、薪資按鈕細項權限、任務頁獨立權限碼、員工/部門/門市管理限 admin+、鎖定/解鎖班表改權限碼（schedule.lock）＋權限頁開關
- **RLS 收斂**：員工個資依角色/門市範圍限制、寫入洞收斂；費用申請/出差/經常性費用拿掉「主管全看」洞（收斂到本人/店長/主管鏈）
- **會計科目重整**：38 個科目 + 紅黑（收入/支出）可見度過濾
- **安全修補**：LINE login 開放轉址/身分偽造、XSS、招募模組 schema drift 修復（補回缺失欄位使新增職缺/人力需求單復活）

### 7. LINE / LIFF / 通知
- 任務卡片步驟進度＋發起人、補打卡待簽核卡、步驟啟動卡帶「需完成表單」
- LIFF 補齊編輯/核銷/查詢 RPC（費用、門市報修、假別餘額、業績獎金）
- @mention 鈴鐺常駐 + 通知面板

### 8. 其他

## 三、品質與基建

- **缺陷修復 287 件**：涵蓋簽核推進、跨午夜打卡、薪資級距/加班費、schema drift、RLS 遞迴/洩漏、LINE 卡片格式等
- **DB 漂移偵測**：`db:drift` 快照機制，防止 Studio 直接改函式未回填 migration
- **大型函式安全改法**：以 live 快照為基底、只改目標段、diff 驗證後包 migration（避免整支重寫洗掉邏輯）
- **重構 7 件**：慣例統一、共用元件對齊、over-fetch 優化
